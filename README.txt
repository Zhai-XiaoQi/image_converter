图片格式转换工具

功能：
- 支持输入：JPG、JPEG、PNG、WEBP、BMP、TIFF
- 支持输出：JPG、PNG、WEBP
- JPG 可选渐进式
- 支持质量设置
- 支持保留目录结构
- 支持覆盖已存在文件
- 支持转换成功后删除原图
- 自动生成 conversion_report.txt

使用：
1. 双击“启动图片格式转换工具.bat”
2. 选择文件夹或多个文件
3. 选择输出目录和目标格式
4. 点击“扫描预览”
5. 确认无误后点击“开始转换”

建议：
- 第一次处理重要素材时，不要勾选“成功后删除原图”。
- 需要 Amazon 渐进式 JPG 时，选择 JPG，并勾选“JPG 渐进式”。
- PNG 有透明背景时，转 JPG 会使用“透明背景”颜色铺底，默认白色。
