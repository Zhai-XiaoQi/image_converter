图片格式转换工具 1.4.3

功能：
- 支持输入：JPG、JPEG、PNG、WEBP、BMP、TIFF、HEIC/HEIF
- 支持输出：JPG、PNG、WEBP
- 支持 Amazon 主图 / A+ 桌面图 / 移动 A+ / WebP 网页图预设
- 支持尺寸调整、固定质量、目标体积压缩
- 支持批量重命名、文字水印、Logo 水印
- 支持水印预览编辑：按输出尺寸预览、拖动位置、缩放、不透明度、角度
- 支持预览区缩放
- 自动处理同名不同格式转同一格式时的目标冲突
- JPG 可选渐进式
- 支持保留目录结构
- 支持覆盖已存在文件
- 支持转换成功后删除原图
- 自动生成 conversion_report.txt

使用：
1. 双击“启动图片格式转换工具.bat”
2. 选择文件夹或多个文件
3. 选择输出目录和目标格式
4. 点击“扫描预览”
5. 确认无误后点击底部“开始转换”

建议：
- 第一次处理重要素材时，不要勾选“成功后删除原图”。
- 需要 Amazon 渐进式 JPG 时，选择 JPG，并勾选“JPG 渐进式”。
- PNG/WEBP 有透明背景时，转 JPG 会使用“转 JPG 时背景色”铺底，默认白色。
